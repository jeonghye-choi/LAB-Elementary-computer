## 🌟 

### ✔ 옴의 법칙

V = IR


<br>
<br>

### ✔ 합성 저항

- 직렬 : R = R1 + R2

- 병렬 : 1/R = 1/R1 + 1/R2

<br>
<br>

### ✔ LAB

- 전원공급기 : 직류 전원 공급장치

- 전압v: voltmeter

    저항과 병렬로 연결 -> 저항에 걸리는 전압측정가능

- 전류I: ammeter

    전류와 직렬로 연결

- 저항R: ohmmeter

    저항과 병렬로 연결

- 멀티미터 : 볼트/옴/미터 측정 (디지털, 아날로그)
